# 120-Day Python Subscription Tracker Curriculum

This curriculum combines four learning tracks into one daily `.py` file:

1. Python writing/coding
2. Code reading and tracing
3. Bug spotting and refactor thinking
4. Pytest
5. White-hat/security thinking

## End goal

By Day 120, the target is high-intermediate Python developer ability:
- clean function writing
- confident code reading
- debugging and refactoring
- pytest coverage
- backend-style service logic
- validation and permissions
- security-aware thinking
- readiness to move into FastAPI project work

## How to use

Open the folder in VS Code.

Each day:
1. Open `Day_X.py`
2. Read the teaching section
3. Complete the reading questions
4. Solve the bug spotting section
5. Complete/adjust the writing tasks
6. Run the file with `python Day_X.py`
7. Run tests with `pytest Day_X.py`
8. Paste your answers/work back into ChatGPT for marking

## Adaptive rule

This curriculum is not locked into rigid beginner phases.

If a lesson is easy without help, future lessons should become harder faster.
If a lesson is difficult, future lessons should reinforce the concept before moving on.

The app being built is an in-house Subscription Tracker.
